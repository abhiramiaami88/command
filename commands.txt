git clone <url>                      => create the repsitory to our local workspace.

git status                           => to check whether git has initialized in ur crrent directory.

git add (filename)                   => add the file  "filename" to staging area(to perform commit and push).

git add .                            => add all the files in the directory to the staging area.

git commit -m "msg"                  => to commit that it has been tracked by git.

git push -u origin main              => to push the commited file to the global workspace.

git init                             => to initialize git in a folder
